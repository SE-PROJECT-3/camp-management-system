# 🏕️ Camp Management System

The **Camp Management System** is a web-based platform designed to help humanitarian organizations manage refugee or aid camps efficiently.  
The system supports family registration, resource management, aid distribution, attendance tracking, and reporting across multiple camps.

---

# 🎯 Project Goals
The system aims to:

- Manage family registration in camps
- Track aid distribution fairly
- Prevent duplicate aid allocation
- Manage camp resources
- Monitor attendance
- Provide analytics and reports for administrators

---

# 🏗️ Technical Stack

- **Backend:** Laravel / PHP
- **Database:** MySQL
- **Frontend:** HTML, CSS, JavaScript
- **Architecture:** MVC
- **Standards:** PSR-12 Coding Standards
- **Testing:** Unit & Integration Testing (Target ≥70%)
